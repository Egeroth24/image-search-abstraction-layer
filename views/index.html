Image Search Abstraction Layer

User Stories:
  1. I can get the image URLs, alt text and page urls for a set of images relating to a given search string.
  2. I can paginate through the responses by adding a ?offset=# parameter to the URL.
  3. I can get a list of the most recently submitted search strings.

Example Usage:

https://https://nine-flier.glitch.me/.glitch.me/cute%20kittens
https://https://nine-flier.glitch.me/.glitch.me/latest
